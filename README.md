# 🚙 Toyota Hilux 2025 FiveM Script

Professional and feature-rich FiveM script for Toyota Hilux 2025 vehicle with extensive customization options.

## ✨ Features

### Vehicle Options
- **6 Color Variants**: Black, White, Gray, Red, Blue, Military Green
- - **4 Bed Options**: Empty Bed, Date Crates, Metal & Iron, Wooden Bed
  - - **3 Mirror Options**: Standard, Flat, None
    - - **3 Emblem Options**: Gold, Chrome, None
      - - **3 Exhaust Options**: Chrome, Standard, Sport
        - - **3 Line Options**: 2002 Gulf, 2005 Saudi, None
          - - **6 Extras**: Front Plate, Rear Plate, Side Bars, Side Protection, Extra Lights, Enhanced Sound System
           
            - ### Vehicle Specifications
            - - **Model**: Toyota Hilux 2025
              - - **Price**: $85,000
                - - **Top Speed**: 180 km/h
                  - - **Category**: Off-Road Trucks
                    - - **Weight**: 1800 kg
                      - - **Tank Capacity**: 80 liters
                       
                        - ## 📋 Commands
                       
                        - | Command | Description |
                        - |---------|-------------|
                        - | `/spawhhilux` | Spawn a Toyota Hilux 2025 |
                        - | `/customhilux [number]` | Change vehicle color (1-6) |
                        - | `/hiluxinfo` | Display vehicle information |
                        - | `/fixhilux` | Repair the vehicle |
                       
                        - ## 🚀 Installation
                       
                        - 1. **Clone the Repository**
                          2. ```bash
                             git clone https://github.com/a1064074535-droid/toyota-hilux-2025-fivem.git
                             ```

                             2. **Place in Resources Folder**
                             3. ```
                                FiveM Server/
                                └── resources/
                                    └── toyota-hilux-2025-fivem/
                                ```

                                3. **Add to server.cfg**
                                4. ```
                                   ensure toyota-hilux-2025-fivem
                                   ```

                                   4. **Restart Server**
                                  
                                   5. ## 📁 File Structure
                                  
                                   6. ```
                                      toyota-hilux-2025-fivem/
                                      ├── fxmanifest.lua      # Resource manifest
                                      ├── config.lua          # Configuration and settings
                                      ├── client.lua          # Client-side script
                                      ├── server.lua          # Server-side script
                                      ├── locales.lua         # Language support (AR/EN)
                                      └── README.md           # This file
                                      ```

                                      ## 🎮 Usage

                                      ### Spawn Vehicle
                                      ```
                                      /spawhhilux
                                      ```
                                      Spawns a Toyota Hilux 2025 at your current location.

                                      ### Customize Color
                                      ```
                                      /customhilux 1  (Black)
                                      /customhilux 2  (White)
                                      /customhilux 3  (Gray)
                                      /customhilux 4  (Red)
                                      /customhilux 5  (Blue)
                                      /customhilux 6  (Military Green)
                                      ```

                                      ### Vehicle Info
                                      ```
                                      /hiluxinfo
                                      ```
                                      Displays detailed vehicle information in chat.

                                      ### Repair Vehicle
                                      ```
                                      /fixhilux
                                      ```
                                      Repairs all damage and restores the vehicle.
